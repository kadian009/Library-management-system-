# Library-management-system-

📌 Overview

This project is a console-based Library Management System developed in C++. It helps in managing library records efficiently using file handling and menu-driven operations.
The system allows users to perform basic library operations such as adding books, issuing books, returning books, and viewing records.

🚀 Features

📖 Add new books
🔍 Search book by ID or name
📋 Display all books
📥 Issue books
📤 Return books
❌ Delete book records
💾 File handling for permanent storage
📊 Menu-driven user interface

🛠️ Technologies Used
Language: C++

Concepts Used:

File Handling
Object-Oriented Programming (OOP)
Functions & Loops
Conditional Statements
